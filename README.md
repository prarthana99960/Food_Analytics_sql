# Food Delivery Analytics SQL Project

## Project Overview
This is an industry-level SQL Data Analytics project built using MySQL Workbench on a realistic Food Delivery dataset. The project focuses on analyzing customer behavior, restaurant performance, delivery efficiency, revenue trends, and business KPIs using SQL queries.

---

# Tools & Technologies
- MySQL Workbench
- SQL
-  CSV Dataset

---

# Database Tables
- Customers
- Restaurants
- Orders
- Delivery Agents
- Deliveries

---

# SQL Concepts Used
- Joins
- Aggregate Functions
- GROUP BY & ORDER BY
- CASE Statements
- Date Functions
- Subqueries
- CTEs
- Views
  

---

# Business Analysis Performed
- Revenue Analysis
- Top Customers Analysis
- Restaurant Performance Analysis
- City-wise Revenue Analysis
- Payment Method Analysis
- Delivery Performance Analysis
- Cancellation Rate Analysis
- Monthly Revenue Trends
- Peak Ordering Hours Analysis

---

# Key Insights
- Identified top-performing restaurants and cities
- Analyzed customer spending behavior
- Evaluated delivery efficiency
- Measured cancellation trends
- Generated business KPIs for decision-making
